# Bootstrap

This repo is used to bootstrap my workstation.

## Installation

To bootstrap a new Linux laptop,
```
sudo curl https://raw.githubusercontent.com/acodefella/bootstrap/main/install | bash
```

## Tools

This script installs the following tools,
1. Terraform
2. aws cli
3. azure cli
4. vnc viewer
5. kind
6. kubectl
